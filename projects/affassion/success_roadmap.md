# Success Roadmap: Affassion (The 5 Pillars)

プロジェクト `Affassion` が確実に収益化するための詳細ロードマップです。
「コードを書くこと」以上に「売れる仕組みを作ること」に焦点を当てています。

## 1. 📂 Data Collection Section (商品力)

**目的**: 「ハズレ」を排除し、ユーザーに確実な「アタリ」だけを届ける。

*   **Phase 1: Human Curation (手動厳選)**
    *   [ ] **高評価フィルター**: SHEIN内で「レビュー数1000以上」「評価4.8以上」のアパレルのみを抽出。
    *   [ ] **実写レビュー確認**: ユーザー投稿画像を確認し、「写真詐欺（生地がペラペラ等）」がないアイテムだけを選定。
    *   [ ] **カテゴリ集中**: 最初は「ワンピース」や「オフィスカジュアル」など、1つのジャンルに絞る（専門性を出すため）。

## 2. 🎨 Design Section (ブランド力)

**目的**: 「SHEIN＝安っぽい」という先入観を、「Affassion＝賢い買い物」という体験に変える。

*   **Tooling (効率化)**
    *   [x] **Google Stitch Prototyping**: ビジュアルデザインとHTML/CSSのベース作成には **Google Stitch** を活用し、最速で理想の「雑誌風UI」を固める。
    *   [ ] **Antigravity Integration**: Stitchで確定したデザインを、Next.js + Tailwind CSS のコンポーネントとして実装コードに変換する。
*   **Visual Identity (見た目)**
    *   [ ] **"Vogue" Style UI**: 雑誌のようなレイアウト。大きな高品質画像と、余白を活かしたタイポグラフィ。
*   **Search & UX (検索体験)**
    *   [ ] **"Purpose" Search**: 「デート」「結婚式二次会」など、着用シーンで探せる検索軸。

## 3. 💎 Differentiation Section (差別化)

**目的**: 本家SHEINではなく、わざわざ当サイト経由で買う理由を作る。

*   **Editorial Cuts (切り口)**
    *   [ ] **"High-Look" Guarantee**: 「1000円台に見えない」アイテムだけを集めた特集。
    *   [ ] **Fear Resolution**: 「160cm/50kgならMが正解」といった具体的なサイズガイド。

## 4. 📢 Awareness Section (認知・集客)

**目的**: 適切なターゲットにリーチし、サイトへ誘導する。

*   **SNS Marketing (瞬発力)**
    *   [ ] **Instagram/TikTok**: ショート動画でバズを狙う。
    *   [ ] **Pinterest**: カタログ画像を大量にピンして流入を稼ぐ（資産型）。
*   **SEO (持久力)**
    *   [ ] **Long-tail Keywords**: 悩み系キーワードでのブログ記事作成。

## 5. 🛡️ Trouble Avoidance Section (トラブル回避)

**目的**: ビジネスを持続可能にするための守り。

*   **Legal Compliance (法律)**
    *   [ ] **商標権**: ドメインや広告文に「SHEIN」という単語を直接使用しない。
    *   [ ] **景品表示法**: ステマ規制対応の表記を行う。
