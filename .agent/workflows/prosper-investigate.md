---
description: Prosper Investigator - テーマを指定して徹底的な調査を行い、レポートを作成する
---

ユーザーが「○○についての調査をお願い」「○○の記事を書きたいので調べて」などと依頼した場合、このワークフローを使用してください。

1. **テーマの特定**: ユーザーの依頼から調査対象のテーマ（例: "映画 マトリックス"）を特定してください。
2. **コマンドの実行**: 以下のコマンドを実行して調査を開始します。

```bash
# テーマを引数として渡す
projects/prosper/.venv/bin/python projects/prosper/investigator/investigator.py "{Theme}" --ai-plan
```

3. **完了報告**:
   - コマンドが完了すると、レポートファイルのパスが表示されます。
   - ユーザーに「調査が完了しました。レポートはこちらです：[ファイルパス]」と報告してください。

### 注意事項
- 実行には数分かかる場合があります。
- API制限（429エラー）が発生した場合、自動的に軽量モデル（Lite）に切り替わって続行されるため、そのまま待機してください。
